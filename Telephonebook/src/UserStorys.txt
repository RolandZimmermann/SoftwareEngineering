UserStorys TelefonBook:

Als User möchte ich alle Einträge des Adressbuchs einsehen.
Als User möchte ich neue Einträge anlegen.
Als User möchte ich vorhandene Einträge löschen.
Als User möchte ich vorhandene Einträge bearbeiten.
Als User möchte ich nach Einträgen komfortabel suchen.

Als User möchte ich Einträge laden.
